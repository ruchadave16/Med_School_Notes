*weak acid + base and conj. salt*
*any substance that takes in or releases $H^+$ to minimize change to pH in body*

#### Bicarbonate Buffer
Most important in [[Extracellular Matrix|ECM]] fluid

>[!Equation] $H_2O + CO_2 → H_2CO_3 → H^+ + HCO_3^-$

If acid added
- $HCl + NaHCO_3 -> H_2CO_3 + NaCl$
- ↓ $HCO_3^-$ 

If base added
- $NaOH + H_2CO_3 → NaHCO_3 + H_2O$
- ↓ $H_2CO_3$ 

[[Henderson-Hasselbalch Equation]] 

>[!Equation] $pH = pK + log \frac{HCO_3^-}{H_2CO_3}$

- Here $[H_2CO_3]$ = $H_2CO_3$ + dissolved $CO_2$
- Can ignore $H_2CO_3$ since it’s so low

#### Phosphate Buffer
- pK = 6.8 and pH of body = 7.4 
- Better buffer but lower concentration so not effective
- Useful in renal tubule → ↑ concentration, pH more acidic, accepts $H^+$ secreted into tubular lumen
