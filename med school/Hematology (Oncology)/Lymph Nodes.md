 *Bean-shaped glands along lymphatic vessels that filter [[lymph]]*

**Sentinel node** – first node to receive lymph
Encapsulated lymphoid tissue
#### Function
- Drain fluid from site of infection
	- [[Langerhans Cells|Dendritic cells]] activated → express [[MHC-I]], [[MHC-II]], B7
	- These enter lymph carrying processed antigens and free antigens
- Lymph enters nodes where B and T cells find matching antigen
- Dendritic cells present to T cells
- Antigen presenting cells process antigen
- B cells react to antigen
- Generate [[Adaptive Immunity|adaptive immunity]] response
#### Structure

![[Drawing 2026-01-19 01.27.00.excalidraw]]

- Capsule: fibrous connective tissue with trabeculae
- Stroma: [[Reticular Fibers|reticular fiber]] – Type III [[Collagen|collagen]]
- Parenchyma: lymphocytes, [[Antigen Presenting Cells|APC]], granulocytes
	- Cortex: lymph nodules with [[Lymphoid Follicles]], mainly [[B Lymphocytes|B cells]]
	- [[Lymphoid Paracortex]]: between cortex and medulla, mainly [[T Lymphocytes|T cells]]
	- Medulla: cords (with [[Plasma Cells|plasma cells]]) that branch between medullary sinuses (containing [[Macrophage|macrophages]] which filter lymph through [[Phagocytosis]])
- Afferent Lymphatic Vessel: pierces convex margin of capsule → sub-capsular sinuses → cortical sinuses → medullary sinuses → efferent lymphatic vessels
- Efferent Lymphatic Vessel: exits hilum

![[Pasted image 20250817172721.png]]

- [[Blood Vessels]]
	- Enter and leave at hilus
	- Lymphocytes exit into nodes through ‘high endothelial’ venules (cuboidal epithelial cells)
