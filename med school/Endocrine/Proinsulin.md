*Precursor peptide to [[insulin]]*

- Single peptide strand with connecting portion between A and B chains called C peptide

![[Pasted image 20251027212736.png]]

- Packaged into secretory granules, C peptide cleaved out making [[insulin]]
- C peptide and insulin released together
	- C peptide has no function, but can measure beta cell function

>[!Clinical]
>C peptide can be used to track [[Type 1 Diabetes]]
>Can also confirm exogenous insulin (medicinal insulin DOESN’T have this)
