Cell-mediated reaction
- **No antibodies**
- Memory [[T Lymphocytes|T cells]]
	- Activate Th1 for macrophages and Th2 for eosinophils
#### Example
- [[Tuberculosis]] Test PPD
	- Tuberculin injected into skin
	- Previous exposed person has memory T cells 
	- CD4 recognice protein on MHC II
	- [[Th1]] response
	- Macrophage response so redness 24-72 hrs later
- Immune response to mycobacteria, fungi
- [[Contact dermatitis]]
	- Chemicals attach to skin cells
	- CD8 cells attack skin cells
	- Erythema + itching 12-48 hrs later
- [[Multiple Sclerosis]]
