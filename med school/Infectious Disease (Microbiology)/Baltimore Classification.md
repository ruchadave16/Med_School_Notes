- [[mRNA]] synthesis = first step of viral gene expression
- Genome can be both DNA or RNA
	- RNA can be positive sense (5' - 3' orientation) or negative sense (3' - 5')
	- (+) is much more 
#### Class I: ds DNA
- mRNA transcribed from dsDNA through DNA dependent RNA polymerase
#### Class II: ss DNA
- ssDNA used as template to make dsDNA
- mRNA transcribed from dsDNA through DNA dependent RNA polymerase 
#### Class III: ds RNA
- One strand is + and one strand is - 
- + can function as mRNA directly
#### Class IV: ss (+) RNA
- Can act directly as mRNA and be translated inside of host
	- Use RNA dependent RNA polymerase to make mRNA and genomic RNA
	- + sense makes - sense
	- - sense is template for making many more + sense RNA
		- Some become mRNA for protein synthesis
		- Rest are genomic mRNA
#### Class V: ss (-) RNA
- Convert - to + sense to act as mRNA
	- RNA dependent RNA polymerase is found inside virion for this purpose
#### Class VI: [[reverse transcriptase]]
- Start with ssRNA and use reverse transcriptase to make ssDNA
- RNA degraded by nucleases in host
- Make another DNA strand from the template to form dsDNA
- Integration of dsDNA into host genome
- Host has DNA dependent RNA polymerase that transcribes viral mRNA
#### Class VII: Reverse Transcriptase
- start with partial DNA and convert to dsDNA
- This enters host nucleus and is converted by DNA dependent RNA polymerase to mRNA and pre-genomic RNA
- pgRNA converted back to DNA by reverse transcriptase
