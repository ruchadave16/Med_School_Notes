- Polysaccharide antigen
	- No T-cell activation because T-cell independent [[B Lymphocytes|B-cell]] activation
	- Poor memory
	- Weak response but weak protection

- Conjugation to peptide antigen
	- B-cells present antigen to T-cells so stronger response and better memory

>[!Info] Examples
>H. Influenza type B
>Neisseria meningitidis
>Streptococcus pneumoniae
